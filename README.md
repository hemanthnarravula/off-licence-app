# Off-licence App

Multi-store UK off-licence mobile app (Expo) with a Next.js API on Vercel and Neon Postgres.

See [PLAN.md](./PLAN.md) for the full product and technical plan.

## Stack

- **Mobile:** Expo (iOS + Android)
- **Backend:** Next.js on Vercel
- **Database:** Neon Postgres
- **Auth:** Better Auth

## Status

Planning phase — implementation not started yet.
